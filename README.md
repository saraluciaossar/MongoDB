# MongoDB

This repository contains a set of exercises developed during the MongoDB module of the IT Academy bootcamp. The goal is to practice working with NoSQL databases using real-world inspired data related to a movie platform.

### Dataset

The database is composed of several collections that simulate a cinema application:

- users: user information (name, email, etc.)
- movies: movie details (title, genre, duration, rating, directors, etc.)
- comments: user comments on movies
- theatres: cinema locations with geographic data
- sessions: user sessions and authentication data

### Objectives
- Understand how NoSQL databases work
- Practice querying documents in MongoDB
- Work with nested structures and arrays
- Perform aggregations and filtering
- Explore relationships between collections

### Key Concepts Practiced
Basic queries using find()
Filtering with operators ($match, $gt, $in, etc.)
Aggregation framework ($group, $project, $sort)
Working with embedded documents and arrays
Data exploration and transformation

### Tools Used
MongoDB Compass
